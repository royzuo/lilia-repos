# Cinematic Prompt Formula

Use this formula for all video generation tasks.

`[Camera] + [Subject] + [Action] + [Atmosphere] + [Lighting] + [Quality Tags]`

## Example
**Input**: "Qin Terracotta Army"
**Optimized**: "[Camera]: Low-angle cinematic tracking shot, slowly moving through the infantry columns. [Subject]: Thousands of life-sized terracotta soldiers standing in rigid formation. [Action]: Dust particles dancing, absolute stillness. [Atmosphere]: Solemn, epic, ancient. [Lighting]: Dramatic chiaroscuro lighting, warm golden rays. [Quality Tags]: Cinematic texture, hyper-realistic, authentic Qin armor aesthetic."
