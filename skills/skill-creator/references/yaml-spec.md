# YAML Metadata Spec

Every skill must have a `SKILL.md` with YAML frontmatter:

```yaml
---
name: your-skill-name
description: A clear description of what this skill does and when to use it
---
```

- `name`: Unique identifier (lowercase-hyphen-case).
- `description`: The *most important* part. Must be comprehensive and include all "when to use" trigger context.
